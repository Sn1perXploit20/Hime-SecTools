# Hime-SecTools
simple script to get 300++ Special Security tools 

## How to use?
**Download with git**
`git clone https://https://github.com/Sn1perXploit20/Hime-SecTools.git`
**change directory**
`cd Hime-SecTools`
**Run**
`bash hime.sh`
